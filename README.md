# Download Requirements
### Type: 
    source venv/bin/activate
### This should give you the ability to see (venv) next to your name in terminal.  Make sure you are using python3.10.6.
    python3 --verion

    pip3 install -r requirements.txt
### Now run main.py:
    python3 main.py
